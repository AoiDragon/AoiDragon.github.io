C++编程练习 MyString

类
C++基础
发布日期:   2020-03-23

描述

补足MyString类，使程序输出指定结果

#include <iostream>
#include <string>
#include <cstring>
using namespace std;
class MyString
{
    char *p;

public:
    MyString(const char *s)
    {
        if (s)
        {
            p = new char[strlen(s) + 1];
            strcpy(p, s);
        }
        else
            p = NULL;
    }
    ~MyString()
    {
        if (p)
            delete[] p;
    }
    // 在此处补充你的代码
};
int main()
{
    char w1[200], w2[100];
    while (cin >> w1 >> w2)
    {
        MyString s1(w1), s2 = s1;
        MyString s3(NULL);
        s3.Copy(w1);
        cout << s1 << "," << s2 << "," << s3 << endl;

        s2 = w2;
        s3 = s2;
        s1 = s3;
        cout << s1 << "," << s2 << "," << s3 << endl;
    }
}

样例输入

abc def
123 456

样例输出

abc,abc,abc
def,def,def
123,123,123
456,456,456

首先分析我们需要对类添加哪些功能

  • MyString s1(w1) 需要构造函数，题目已经给出
  • s2 = s1需要复制构造函数（这里的=用于初始化而非赋值）
  • s3.Copy(w1)需要自己编写Copy函数
  • <<需要重载
  • s3=s2需要对=重载

s2=w2这条语句会由默认类型转换构造函数把w2转化为临时对象，再通过重载过的=赋值给s2，所以不需要我们编写。

复制构造函数

MyString(MyString& a)
{
    if(a.p)
    {
        p=new char[strlen(a.p)+1];
        strcpy(p, a.p);
    }
    else
    {
        p=NULL;
    }
}

复习：复制构造函数起作用的三种情况

  • 用一个对象去初始化同类的另一个对象
  • 某函数的一个参数是类A的对象，则函数被调用时类A的复制构造函数被调用
  • 函数的返回值是类A的对象，则函数返回时类A的复制构造函数被调用

Copy函数

void Copy(const char *s)
{
    if (p)
        delete[] p;//防止空间浪费
    if (s)
    {
        p = new char[strlen(s) + 1];
        strcpy(p, s);
    }
    else
         p = NULL;
}

重载<<

friend ostream &operator<<(ostream &o, MyString a)
{
    cout << a.p;
    return o;
}

流插入运算符的返回值是ostream&，所以参数列表应该有两个参数，不能直接在类的内部声明。可将其声明成MyString的友元函数并在类内部定义。

重载=

MyString &operator=(const MyString &a)
{
    if (p == a.p)
        return *this;
    else
    {
        delete[] p;
        p = new char[strlen(a.p) + 1];
        strcpy(p, a.p);
        return *this;
    }
}

a必须声明为const类型，因为对于语句s2=w2,需要先通过类型转换函数将w2转化为临时对象，而临时对象均为const类型。
