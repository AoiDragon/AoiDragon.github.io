HDU 2159 FATE

dp 冬训补题
ACM题解
发布日期:   2020-06-17

题目传送门

题目描述

最近xhd正在玩一款叫做FATE的游戏，为了得到极品装备，xhd在不停的杀怪做任务。久而久之xhd开始对杀怪产生的厌恶感，但又不得不通过杀怪来升完这最后一级。现在的问题是，xhd升掉最后一级还需n的经验值，xhd还留有m的忍耐度，每杀一个怪xhd会得到相应的经验，并减掉相应的忍耐度。当忍耐度降到0或者0以下时，xhd就不会玩这游戏。xhd还说了他最多只杀s只怪。请问他能升掉这最后一级吗？

输入数据有多组，对于每组数据第一行输入n，m，k，s(0 < n,m,k,s < 100)四个正整数。分别表示还需的经验值，保留的忍耐度，怪的种数和最多的杀怪数。接下来输入k行数据。每行数据输入两个正整数a，b(0 < a,b < 20)；分别表示杀掉一只这种怪xhd会得到的经验值和会减掉的忍耐度。(每种怪都有无数个)

输出升完这级还能保留的最大忍耐度，如果无法升完这级输出-1。

题解

这道题突出一个变量多，导致想递推方程时不知道往哪个维度应该对应哪个变量。由于第一次接触这类问题，所以只能在别人题解的基础上给出浅显的一些理解。

首先$dp[i][j]$代表花费i点耐心值，杀了j个怪物后获得的最大经验值。杀怪数这一维可以类比背包问题，杀怪数量的限制是背包容量，每个怪物都看作体积为1的物品。但第一维为什么选择用耐心值呢？我觉得也是要类比背包问题的思路，背包问题我们都是在符合条件的区间遍历，而本题中耐心值也要在一定范围内。反观经验值的取值，只有下限而没有上限，不适合遍历。

而怪的种类没有必要用数组的一维表示，for遍历一次即可（类比完全背包）。

明确了各个变量的位置和dp数组的含义，递推方程倒是不那么困难了。代码如下。

#include<iostream>
#include<algorithm>
#include<cstring>

using namespace std;

int n, m, k, s, dp[105][105], a[105], b[105];

int main(void)
{
    while(cin>>n>>m>>k>>s)
    {
        memset(dp, 0, sizeof(dp));
        int ans=-1, flag=0;
        for(int i=1; i<=k; i++)
            cin>>a[i]>>b[i];
        for(int i=1; i<=k; i++)
            for(int j=b[i]; j<=m; j++)
                for(int k=1; k<=s; k++)
                    dp[j][k]=max(dp[j][k], dp[j-b[i]][k-1]+a[i]);
        for(int i=0; i<=m; i++)
        {
            for(int j=0; j<=s; j++)
                if(dp[i][j]>=n)
                {
                    ans=m-i;
                    flag=1;
                }
            if(flag)
                break;
        }
        cout<<ans<<endl;
    }
    return 0;
}
