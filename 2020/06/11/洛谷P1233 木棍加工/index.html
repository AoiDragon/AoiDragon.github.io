洛谷P1233 木棍加工

dp 洛谷
ACM题解
发布日期:   2020-06-11

原题链接

一堆木头棍子共有n根，每根棍子的长度和宽度都是已知的。棍子可以被一台机器一个接一个地加工。机器处理一根棍子之前需要准备时间。准备时间是这样定义的：

第一根棍子的准备时间为1分钟；

如果刚处理完长度为L，宽度为W的棍子，那么如果下一个棍子长度为Li，宽度为Wi，并且满足L>＝Li，W>＝Wi，这个棍子就不需要准备时间，否则需要1分钟的准备时间；

计算处理完n根棍子所需要的最短准备时间。比如，你有5根棍子，长度和宽度分别为(4, 9)，(5, 2)，(2, 1)，(3, 5)，(1, 4)，最短准备时间为2（按(4, 9)、(3, 5)、(1, 4)、(5, 2)、(2, 1)的次序进行加工）。

我们有长度、宽度两个变量要处理，看起来挺复杂的。但我们可以通过事先排序减少讨论的复杂度。如果先看长度这一个变量，为了满足题目要求我们肯定希望长度按降序排列。所以我们可以直接将木棍按长度降序排序，再讨论宽度。

我们要在排好序的木棍中找到宽度也依次下降的序列，同时我们还要满足长度的递减。因为长度已经排过序，那么下标较大的木棍长度一定较小。所以问题就转化为求这个序列的不上升（对应宽度递减）子序列（对应长度递减）的个数。

这里还需要介绍Dilworth定理：偏序集能划分成的最少的全序集个数等于最大反链的元素个数。

说人话就是序列的不上升子序列最少划分数等于最长上升子序列的长度。

所以问题又转化为求该序列的最长上升子序列。

#include<iostream>
#include<algorithm>

using namespace std;

struct node{
    int x, y;
}stick[5005];
int n, dp[5005], m, ans;

bool cmp(node a, node b)
{
    if(a.x==b.x)
        return a.y>b.y;
    else
        return a.x>b.x;
}

int main(void)
{
    cin>>n;
    for(int i=1; i<=n; i++)
        cin>>stick[i].x>>stick[i].y;
    sort(stick+1, stick+n+1, cmp);
    for(int i=1; i<=n; i++)
        dp[i]=1;
    for(int i=1; i<=n; i++)
    {
        for(int j=1; j<i; j++)
        {
            if(stick[i].y>stick[j].y)
                dp[i]=max(dp[i], dp[j]+1);
        }
    }
    for(int i=1; i<=n; i++)
        ans=max(ans, dp[i]);
    cout<<ans<<endl;
    return 0;
}
