洛谷P2758 编辑距离

dp 洛谷
ACM题解
发布日期:   2020-06-07

原题链接

设A和B是两个字符串。我们要用最少的字符操作次数，将字符串A转换为字符串B。这里所说的字符操作共有三种：

1、删除一个字符；

2、插入一个字符；

3、将一个字符改为另一个字符；

！皆为小写字母！

一开始我单纯地以为可以求最长公共子序列来解，后来证明我还是太天真。但dp数组表示形式是一样的，用dp[i] [j]表示将a[1…i]修改为b[1…j]这部分字符串所需的最少操作次数。

首先是最简单的情况，如果a[i-1]==b[j-1]，那么这一步我们不需要任何操作。

dp[i][j]=dp[i-1][j-1];

如果不相同，我们有三种选择：删除、插入和改写。每种选择都只对应一步操作。所以我们如果可以将a[0…i]化为b[0…j]，那么我们一定可以得到只需要一步就使它们相同的状态。

  1. 将a[1…i]转化为b[1…j-1]
  2. 将a[1..i-1]转化为b[1…j]
  3. 将a[1…i-1]转化为b[1…j-1]

从这三种状态到现在分别需要如下操作

  1. 在a[1…i]后添加b[j]，对应添加操作

    dp[i][j]=dp[i][j-1]+1;

  2. 将末尾的a[i]删去，对应删除操作

    dp[i][j]=dp[i-1][j]+1;//dp[i-1][j]就是将a[1..i-1]转化为b[1...j]所需的步数

  3. 将a[i]替换为b[j]，对应替换操作

    dp[i][j]=dp[i-1][j-1]+1;

我们只需要求出三个中最小的即可。

#include<iostream>
#include<algorithm>

using namespace std;

string a, b;
int dp[2005][2005], la, lb;

int main(void)
{
    cin>>a>>b;
    la=a.size();
    lb=b.size();
    //dp开始前还要记得初始化边界条件，即有一个字符串为空时
    for(int i=0; i<=la; i++)
        dp[i][0]=i;
    for(int j=0; j<=lb; j++)
        dp[0][j]=j;
    for(int i=1; i<=la; i++)
    {
        for(int j=1; j<=lb; j++)
        {
            if(a[i-1]==b[j-1])
                dp[i][j]=dp[i-1][j-1];
            else
                dp[i][j]=min(min(dp[i][j-1], dp[i-1][j]), dp[i-1][j-1])+1;
        }
    }
    cout<<dp[la][lb]<<endl;
    return 0;
}
