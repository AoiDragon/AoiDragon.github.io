《挑战程序设计竞赛》——Layout-POJ-3169

无标签
发布日期:   2020-04-06

题目描述

原题链接

思路

记第i头牛的位置为d[i]，则有d[AL]+DL>=d[BL]，d[AD]+DD<=d[BD]。这些不等式的两边都只出现了一个变量(指两头牛之间的距离)。这种特殊形式的不等式方程组称为差分约束系统。附上一篇较为详细的博客

而通过对比差分约束和最短路问题，我们可以发现他们在表达形式上是相同的。在最短路问题中，设起点为s，到各顶点的距离为d[i] (i=u、v……）对于每条权值为w的边e=(v, u),都有d[v]+w>=d[u]成立。同时，对于满足这些约束不等式的d中，有d[v]-d[s]<=w成立(将上式移项)。即d[v]-d[s]的最大值就是所有w的最小值(w可以不直接连接v和s，看作从v到s的所有路径比较好理解)，也就是从s到v的最短距离。

所以d[AL]+DL>=d[BL]等价于从AL向BL连一条权为DL的边，d[AD]+DD<=d[BD]等价于从BD向AD连一条权为-DD的边(移项)。我们要求d[N]-d[1]的最大值，对应点N到点1的最短路。同时由于图中存在负权边，我们采用SPFA处理。当某个点出列超过点的总数N时，说明图中存在负环。这是因为若没有负环，则任意最短路上的点必定少于N个，最短路共有N-1条，所以如果有一个点的入队次数超过N，就说明图中形成了负环。

代码

自己写的版本，POJ老是CE，正确性不能保证，问题应该不大(?)

#include<cstdio>
#include<cstring>
#include<iostream>
#include<queue>
#include<vector>
#include<algorithm>

using namespace std;

typedef pair<int, int> p;
vector<p> g[1005];
queue<int> q;
int N, ML, MD, flag=0;//ML关系好，MD关系差
int dis[1005], vis[1005], out[1005];

void SPFA()
{
    for(int i=0; i<=N; i++)
        dis[i]=0x3f3f3f3f;
    dis[1]=0;
    vis[1]=1;
    q.push(1);
    while(!q.empty())
    {
        int now=q.front();
        q.pop();
        out[now]++;
        if(out[now]>N)
        {
            flag=1;
            break;
        }
        vis[now]=0;
        for(int i=0; i<g[now].size(); i++)
        {
            pair<int, int> to=g[now][i];
            if(dis[to.first]>dis[now]+to.second)
            {
                dis[to.first]=dis[now]+to.second;
                if(vis[to.first]==0)
                {
                    q.push(to.first);
                    vis[to.first]=1;
                }
            }
        }
    }
    if(flag)
        cout<<-1<<endl;
    else
    {
        if(dis[N]==0x3f3f3f3f)
            cout<<-2<<endl;
        else
            cout<<dis[N]<<endl;
    }
}

int main(void)
{
    cin>>N>>ML>>MD;
    for(int i=0; i<ML; i++)
    {
        int A, B, D;
        scanf("%d%d%d", &A, &B, &D);
        g[A].push_back({B, D});
    }
    for(int i=0; i<MD; i++)
    {
        int A, B, D;
        scanf("%d%d%d", &A, &B, &D);
        g[B].push_back({A, -D});
    }
    SPFA();
    return 0;
}
